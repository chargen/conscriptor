Firmware Upgrades

 

BOB-4 customers can obtain the most recent firmware programming files by making an explicit email request through the Contact form at www.decadenet.com. Please review the comments on firmware upgrade issues in the Troubleshooting section of the latest BOB-4 Application Guide. Firmware installation is possible via BOB-4's main serial port or debug port. Use a terminal program or other means of confirming good communication to BOB-4, and then disconnect the terminal program from the port before proceeding!

Firmware Upgrade Procedure

  1. Apply power to the BOB-4 module.
  2. Select the proper serial port in the Serial-->Port menu.
  3. If you are using the BOB-4 main serial port, then select the currently configured BOB-4 main port baud rate from the Serial-->Baud Rate menu. Default is 9600bps. If you are using the BOB-4 debug port, then this step is not necessary because the debug port is fixed at 115.2kbps.
  4. Select either Serial-->Download-->Firmware (main port) or Serial-->Download-->Firmware (debug port).
  5. When the Open File dialog appears, select the firmware upgrade file.
  6. A progress bar dialog will appear to show file transfer progress.
  7. When the programming operation is complete, BOB-4 resets itself and launches the new firmware.

This procedure may not succeed if BOB-4's firmware has become corrupted. As long as the bootloader in BOB-4 is still operational, new firmware can be installed by using the Serial-->Download-->Firmware (repair) procedure.

Firmware Repair Procedure

This option allows the user to download new firmware even if the current firmware is corrupted and nothing but BOB-4's bootloader is operating. To confirm this condition, connect to either the main or debug serial port at 115.2 kbps using a terminal program. At power up, the BOB-4 bootloader emits: “---- BOB4 boot loader (s: 4.1.2)”

Close the connection to the terminal program before proceeding. In the Conscriptor Serial menu, make sure that the correct PC COM port is selected. The Baud Rate setting doesn’t matter in this case because BOB-4's bootloader operates only at 115.2 kbps. Select Serial-->Download-->Firmware (repair). Select the desired BOB-4 firmware file (B4Vxxx.enc) in the Open File dialog when it appears. When the download is complete, BOB-4 resets itself and launches the new firmware.

USB/RS232 Adapters

USB/RS-232 adapters can cause firmware upgrade problems even if normal communications with BOB-4 have been confirmed. It’s best to use a ‘real’ PC RS-232 serial port (COM port) for communicating with BOB-4. If that’s not possible, try more than one brand of USB adapter. We’ve heard that IOGEAR makes a reliable one, and it has been found at Wal-Mart. Decade Engineering staff hasn’t verified this report. EasySync dual RS-232 adapters have worked well in our shop.

Notes:

Custom fonts are not affected by firmware programming sessions. The new firmware will discover any previously installed custom fonts and make them available for use.

BOB-4's previous configuration settings should not be affected by firmware upgrades, but please review the Troubleshooting section of the BOB-4 Application Guide if problems are encountered.
