Importing Fonts

 

The BOB-4 Conscriptor directly imports only BDF (Bitmap Distribution Format) fonts, which are commonly used with Linux operating systems. Many examples are freely available on the web. The Conscriptor converts BDF fonts to BOB-4's internal font format (.b4f) for editing and saving. No changes are made to the original BDF files. A few example BDF fonts are included in the “BDF Files” subdirectory of the BOB-4 Conscriptor installation directory.


Other font formats must first be converted to BDF before the BOB-4 Conscriptor can import them. Here's a couple of free conversion tools:

otf2bdf (converts TrueType fonts to BDF) http://crl.nmsu.edu/~mleisher/ttf2bdf.html

FontForge (edits and converts many formats) http://fontforge.sourceforge.net/


Example websites for BDF fonts:

http://cvsweb.xfree86.org/cvsweb/xc/fonts/bdf

http://hea-www.harvard.edu/~fine/Tech/x11fonts.html


BDF fonts are often compressed with gzip for distribution (tar.gz suffix). If your current file manager doesn't handle that form of compression, gzip is available here:

http://www.gzip.org/
