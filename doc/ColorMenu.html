These descriptions apply only in the default character rendering mode (65) when BOB-4 displays the font. BOB-4 offers many other rendering options. See m command in the BOB-4 Application Guide for additional details.

Clear

Sets current pixel color to transparent. When operating in local video mode, BOB-4 renders transparent pixels black.

Opaque

Sets current pixel color to opaque. When operating in local video mode, BOB-4 renders opaque pixels white.

SeeThru

Sets current pixel color to halftone. When operating in local video mode, BOB-4 renders halftone pixels grey.

Set Clear

Sets the font editor color set.

Set Opaque

Sets the font editor color set.

Set SeeThru

Sets the font editor color set.

Reset Defaut Colors

Retores the font editor color set to the default green-white-grey.
