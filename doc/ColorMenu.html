Color Menu

Transparent

Sets current pixel color to transparent, which is rendered green in Font Editor. When operating in local video mode, BOB-4 renders transparent pixels black.


White

Sets current pixel color to white.


Halftone

Sets current pixel color to halftone, which is rendered gray in Font Editor. Halftone is disabled for 1bpp fonts (one bit per pixel). Halftone pixels are rendered in 50% video (dark video) when BOB-4 is operating in genlock/overlay mode, but they're rendered gray in local video mode


These descriptions apply only in the default character rendering mode (65) when BOB-4 displays the font. BOB-4 offers many other rendering options. See m command in the BOB-4 Application Guide for additional details.
