Cut

Copies the selected character for pasting and clears all pixels from it. Uses buffer memory, not Windows Clipboard. Paste target must be a character cell of the same dimensions.

 

Copy

Copies the selected character for pasting without clearing it. Uses buffer memory, not Windows Clipboard. Paste target must be a character cell of the same dimensions.

 

Paste

Place previously copied character image in the selected character cell. Pastes from buffer memory, not from Windows Clipboard. Source and target cells must have the same dimensions.

 

Clear

Clears all pixels in the selected font character.

 

Insert

Row: Adds one row to the bottom of the font characters. This command affects all characters in the fontset.

Column: Adds one column to the right side of the currently selected character.

Character: Adds a new blank character at the current position in the fontset.

Outline: Adds a halftone outline to all characters in the fontset. This command is only available with 2bpp fonts.

 

Delete

Row: Removes one row from the bottom of the font characters. This command affects all characters in the fontset.

Column: Removes one column from the right side of the currently selected character.

Selected Character: Removes the currently selected character.

 

Move

Down: Moves the currently selected character image one pixel down. A blank row is added at the top.

Up: Moves the currently selected character image one pixel up. A blank row is added at the bottom.

Right: Moves the currently selected character image one pixel to the right. A blank column is added at the left.

Left: Moves the currently selected character image one pixel to the left. A blank column is added at the right.

 

Pixel Depth ("BPP" or "bpp" is the number of bits per pixel.)

1BPP-->2BPP: Converts the current font from 1bpp to 2bpp.

2BPP-->1BPP: Converts the current font from 2bpp to 1bpp. Halftone pixels are converted to transparent.

 

Font Name

Edit the Font Name field of the current font (not the file name).
